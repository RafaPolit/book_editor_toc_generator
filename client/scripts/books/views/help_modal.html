Ayuda

  • El Título es requerido para poder guardar una publicación
  • Ubique el cursor sobre el para arrastrar un ítem
  • Ubique el cursor sobre el contenido para editar () o eliminar ()

  • Usar el teclado es la manera más rápida de crear contenidos.
    Utilice Tab para indentar hacia la derecha.
    Utilice Shift + Tab para hacerlo hacia la izquierda.
    Oprima Enter para agregar el contenido al índice.
